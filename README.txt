# AI-Powered Lead Generation System

An intelligent lead generation system developed as part of the **SkillWallet Group Project**.
The system automatically discovers and qualifies B2B leads using **web scraping, AI analysis, and a user-friendly dashboard**.

It combines **Bright Data scraping services**, **OpenAI-powered analysis**, and an **interactive Streamlit interface** to help users quickly identify and engage potential business prospects.

</div>

---

# Project Overview

The **AI-Powered Lead Generation System** is designed to automate the process of identifying potential business leads. Traditional lead generation requires manual research and filtering, which is time-consuming and inefficient.

This system solves that problem by combining **automated web data collection with AI-based lead qualification**, enabling users to quickly find relevant contacts and receive outreach recommendations.

The platform allows users to describe their target customer in natural language, automatically extract relevant filters, gather leads from online sources, analyze them using AI, and present structured results through an interactive dashboard.

---

# Project Development Phases

## 1. Requirement Analysis & Planning

Requirement Analysis & Planning involves identifying the goals and requirements of the lead generation system before development begins.

During this phase, the team analyzed stakeholder needs, defined the functional and non-functional requirements, and determined the system architecture. The scope included automated lead scraping, AI-based lead qualification, and an interactive dashboard for managing results.

The planning stage also involved evaluating technical feasibility, identifying potential risks, defining timelines, and allocating resources. This structured planning ensured the system would efficiently generate high-quality leads and support business prospecting activities.

---

## 2. Backend Development & Configuration

The backend of the system is responsible for collecting, processing, and analyzing lead data.

Key backend components include:

* Integration with **Bright Data APIs** to collect fresh B2B leads from online sources.
* AI-based analysis using **OpenAI models** to evaluate lead relevance and provide summaries.
* Data processing pipelines that transform raw scraped data into structured lead profiles.
* Configuration of API connections and data workflows.

This backend architecture ensures reliable lead extraction and intelligent qualification.

---

## 3. UI/UX Development & Customization

A **Streamlit-based dashboard** was developed to provide a simple and interactive interface for users.

The interface allows users to:

* Enter lead search queries in natural language.
* View enriched lead profiles with scores and summaries.
* Access outreach recommendations generated by AI.
* Adjust system settings such as number of leads and model selection.

The goal of the UI/UX design is to provide a **clear and intuitive experience** for managing lead generation workflows.

---

## 4. Data Migration, Testing & Security

Testing ensures the system performs accurately and reliably.

Key activities include:

* Verifying correct lead extraction from external sources.
* Testing AI enrichment and scoring accuracy.
* Ensuring API integrations function properly.
* Implementing secure management of API keys using environment variables.
* Validating system performance and error handling.

Security measures were implemented to protect sensitive credentials and ensure safe communication with external APIs.

---

## 5. Deployment, Documentation & Maintenance

The final stage involved deploying the application and documenting the system architecture and workflow.

Deployment steps include:

* Running the application through Streamlit.
* Configuring API keys through environment variables.
* Providing user documentation for operating the system.

Ongoing maintenance includes monitoring system performance, updating dependencies, fixing bugs, and improving AI lead qualification accuracy.

# Installation

Clone the repository:

```bash
git clone <your-repository-url>
cd ai-lead-generator
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

Windows

```bash
venv\Scripts\activate
```

macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# Configuration

Create a `.env` file and add your API credentials:

```
OPENAI_API_KEY=your_openai_api_key
BRIGHT_DATA_API_KEY=your_bright_data_api_key
```

---

# Running the Application

Start the application using Streamlit:

```bash
streamlit run app.py
```

Open your browser and navigate to:

```
http://localhost:8501
```
